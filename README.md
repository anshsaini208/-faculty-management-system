# 🎓 Faculty Management System

A modern and scalable Faculty Management System designed for colleges and universities to efficiently manage faculty attendance, workload, availability, analytics, and reporting.

## ✨ Features

- 🔐 Role-Based Authentication (Dean/Admin & Faculty)
- 📅 Faculty Attendance Management
- 📊 Faculty Workload Analytics
- 👨‍🏫 Faculty Availability Tracking
- 🏢 Department-Wise Analytics
- 📈 Interactive Dashboards & Charts
- 📄 PDF, Excel & CSV Report Export
- 🌙 Dark Mode Support
- 📱 Fully Responsive Design
- ⚡ Optimized Performance with Smooth UX
- 🔄 Real-Time Updates & Notifications

## 🛠️ Tech Stack

### Frontend
- React.js
- TypeScript
- Tailwind CSS
- ShadCN UI
- Framer Motion
- Recharts

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL

### Authentication
- JWT Authentication
- Role-Based Access Control

## 🎯 Objective

To provide a centralized platform for educational institutions to monitor faculty attendance, manage workloads, track availability, and generate actionable insights through powerful analytics dashboards.
